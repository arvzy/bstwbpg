About BESTIE


Introducing BESTIE, The Basic Electronic Stationary Terminal for Indoor Exploration. This application software aims to guide the users (students, school personnel, visitors, etc.) to their destination whilst being stationary.


Features of BESTIE

Building Overview

Users can view the whole layout of the floors using the Building Overview button, having the option to change which floors to view using the buttons below the screen.

Search Facilities

Users have the option to search for a specific facility inside of the building and an animation will play, showing the direction towards the searched facility.

Emergency Exit Plans

It also has the Emergency Exit plans to orient the students on which possible exits the students can take during unfortunate disasters.
